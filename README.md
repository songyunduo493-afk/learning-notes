# learning-notes
Computer Basic Learning Notes & Course Materials
Overview
This repository stores sorted learning resources for college computer basic courses, including classroom courseware, self-organized notes, chapter summaries and review exercises. It aims to record learning progress, organize fragmented knowledge and provide efficient review materials for daily study and exams.
Content & Specification
Materials cover core computer basic knowledge: computer hardware and software systems, operating systems, basic computer networks, information technology and preliminary programming logic. All files are categorized by chapters, updated synchronously with course progress and optimized continuously to fix knowledge blind spots. This repo helps build a solid basic computer knowledge system and improve learning efficiency.
